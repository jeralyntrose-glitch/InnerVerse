**Replace with:**

```css
/* Dark Theme (Default) - MATCHES PHASE 7.1/7.2 */
[data-theme="dark"] {
    /* Backgrounds - Pure black/dark gray (easier on eyes!) */
    --bg-primary: #1a1a1a;      /* Main background - pure dark gray */

    --text-secondary: #b0b0b0;  /* Secondary text - light gray */
    --text-muted: #808080;      /* Muted text - medium gray */
    
    /* Borders - Subtle separation */
    --border-color: #3d3d3d;    /* Default borders */
    --border-hover: #4d4d4d;    /* Hover borders */
    
    /* Accent - Clean blue */
    --accent-primary: #4d9fff;  /* Primary accent - clean blue */
    --accent-hover: #3d8fef;    /* Hover state - darker blue */
    --accent-light: rgba(77, 159, 255, 0.1); /* Light accent bg */
    
    /* Status Colors */
    --success: #28a745;         /* Green - success/complete */
    --success-bg: rgba(40, 167, 69, 0.1);
```
