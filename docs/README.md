2. Follow the template structure
3. Include code examples

## Structure

- **Problem** - What was broken
- **Root Cause** - Why it happened
- **Solution Pattern** - How to fix it (with code)
- **Key Learnings** - Takeaways for future reference
- **Related Files** - Where the fix was applied
